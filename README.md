# node-socketio-vue-chat
Простой онлайн чат, написанный на Node.js, Vue.js, Express.js и Socket.io  
Онлайн-версия: http://165.22.193.68:3000/  

Для установки:
1) npm i
2) node server.js
3) готово! http://localhost:3000/
